# VMB Widgets

A lightweight collection of custom widgets and functionality for Vacation Myrtle Beach.

## Features

- **Reviews Management** - Display and manage customer reviews
- **Specials System** - Create and showcase special offers with categories
- **Custom Shortcodes** - Easy-to-use shortcodes for content display
- **ACF Integration** - Advanced Custom Fields support for flexible content
- **Admin Interface** - User-friendly dashboard for managing widgets and content
- **Responsive Design** - Mobile-friendly widgets and layouts

## Installation

1. Upload the plugin files to `/wp-content/plugins/vmb-widgets`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure settings in the VMB Widgets admin menu

## Usage

**Adding Specials:**
1. Navigate to VMB Widgets > Specials in the WordPress admin
2. Create new specials with custom content and images
3. Assign categories to organize your specials

**Displaying Content:**
Use the built-in shortcodes to display content anywhere on your site. Access the shortcode documentation through the plugin admin interface.

**Managing Reviews:**
Configure review settings through the VMB Widgets admin panel to display customer testimonials across your site.

## Requirements

- WordPress 5.0+
- PHP 7.2+
- Advanced Custom Fields (recommended)
